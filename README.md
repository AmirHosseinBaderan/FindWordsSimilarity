### Data base queries 

#### Create data base and table 
```T-SQL
CREATE DATABASE GroupsDb

CREATE TABLE ProductCategories (
    Id INT PRIMARY KEY IDENTITY(1,1),
    CategoryName NVARCHAR(255)
);
```
